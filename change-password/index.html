Change Password

Click the Reset Password link below (opens in a new window):
https://rdpbroker.cornwall.ac.uk/RDWeb/Pages/en-US/password.aspx
If you are a student enter CCGSTUDENTS\USERNAME in the account name box. If you are staff enter CCG\USERNAME in the account name box. Enter the password you have been given in the current password box. Enter your new password twice. The new password must be at least 8 characters long and must include at least one each of the following:

  • Uppercase letter
  • Lowercase letter
  • Number
  • It must not include any part of your name
  • Click Submit - you should get a message saying password successfully changed.

If you have any problems, please call the Helpdesk on 01209 616111.
