If you are a student enter ccgstudents\username in the account name box.
If you are staff enter ccg\username in the account name box.
Enter the password you have been given in the current password box. Enter your new password twice.

Password requirements:

  • At least 8 characters long
  • Uppercase letter
  • Lowercase letter
  • Number
  • It must not include any part of your name
  • Click Submit - you should get a message saying password successfully changed.
